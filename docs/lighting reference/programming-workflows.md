# Lighting Programming Workflows

Common approaches to building and running a lighting show.

---

## Busking (Live Operation)

**Definition**  
Operating lights live, reacting to music or performers.

**Key Tools**
- Groups
- Palettes
- Effects
- Faders

**Advantages**
- Flexible
- Expressive

**Challenges**
- Requires practice
- Higher operator load

---

## Pre-Programmed Shows

**Cue Stack Based**
- Sequential cues
- Fixed timing

**Timecode Based**
- External synchronization
- Frame-accurate playback

---

## Hybrid Approach

**Structure**
- Timecoded base looks
- Manual effects layered on top

**Common in**
- Concerts
- Theatre with improvisation

---

## Building a Show (Typical Order)

1. Patch fixtures
2. Create groups
3. Build palettes
4. Design base looks
5. Add effects
6. Assign playbacks
7. Rehearse transitions

---

## Chases & Effects

**Chases**
- Step-based
- Rhythm-oriented

**Effects**
- Continuous modulation
- Parameter-based

---

## Palettes Philosophy

**Why Palettes Matter**
- Consistency
- Speed
- Global updates

**Common Palette Types**
- Color
- Position
- Beam
