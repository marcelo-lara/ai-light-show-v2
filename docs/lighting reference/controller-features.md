# Common Lighting Controller Features

This document describes features commonly found across lighting consoles and software,
regardless of brand or manufacturer.

---

## Core Control

**Fixture Patching**  
Assign fixtures to universes, addresses, and modes.

**Live Output**  
Immediate control of fixtures without recording.

**Edit Mode**  
Programming without affecting live output.

---

## Selection & Organization

**Groups**  
Fast fixture selection (by type, position, role).

**Layouts / Views**  
Visual representations of fixtures and controls.

**Fixture Profiles**  
Manufacturer-provided definitions of channel behavior.

---

## Programming Tools

**Palettes / Presets**  
Reusable values for color, position, beam, etc.

**Cue Recording**  
Capture current state as a cue.

**Tracking vs Non-Tracking**  
Whether values persist across cues unless changed.

**Cue Timing Control**  
Independent fade, delay, and follow times.

---

## Playback & Control

**Faders / Playbacks**  
Manual intensity or cue control.

**Buttons / Executors**  
Trigger cues, effects, or macros.

**Master Controls**  
Grand Master, Blackout, Speed Master.

---

## Effects & Automation

**Effects Engine**  
Generate waves, chases, random motion, color effects.

**Rate / Size / Phase**  
Parameters controlling effect behavior.

**Relative vs Absolute Effects**  
Effects that modify existing values vs replace them.

---

## Show Management

**Scenes / Pages**  
Organize playbacks by song or section.

**Macros**  
Predefined command sequences.

**Backup & Redundancy**  
Session save, show files, backup consoles.

---

## Integration

**MIDI Input**  
Trigger cues or control parameters.

**OSC**  
Network-based control protocol.

**Timecode Input**  
Synchronization with external media.
